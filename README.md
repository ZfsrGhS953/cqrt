# cqrt
This file can be added to any Node.js project. When called it will store all the square root values in the memory, and can be called. Works much faster than the Math.sqrt() function.

## Example
```javascript
var _cqrt = require('PATH');
cqrt = new _cqrt();
cqrt.sqrt(num);
```
